Section VEL_CONTROL

  • Setup parameters to control the velocity during a BAND MD run.
  • Section path: CP2K_INPUT / MOTION / BAND / OPTIMIZE_BAND / MD / VEL_CONTROL
  • This section cannot be repeated.

  • Subsections

    • none

  • Keywords

    • ANNEALING
    • PROJ_VELOCITY_VERLET
    • SD_LIKE

  • Keyword descriptions

    • ANNEALING
    ANNEALING {Real}
    Specify the annealing coefficient
    This keyword cannot be repeated and it expects precisely one real.
    Default value: 1.00000000E+000
    • PROJ_VELOCITY_VERLET
    PROJ_VELOCITY_VERLET {Logical}
    Uses a Projected Velocity Verlet instead of a normal Velocity Verlet. Every time the cosine between velocities and forces is < 0 velocities are zeroed.
    This keyword cannot be repeated and it expects precisely one logical.
    The lone keyword behaves as a switch to .TRUE.
    Default value: .TRUE.
    • SD_LIKE
    SD_LIKE {Logical}
    Zeros velocity at each MD step emulating a steepest descent like(SD_LIKE) approach
    This keyword cannot be repeated and it expects precisely one logical.
    The lone keyword behaves as a switch to .TRUE.
    Default value: .FALSE.
