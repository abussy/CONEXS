Section CEPERLEY

  • Enables sampling with Ceperley's algorithm
  • Section path: CP2K_INPUT / MOTION / PINT / HELIUM / CEPERLEY
  • This section cannot be repeated.

  • Subsections

    • M-SAMPLING

  • Keywords

    • BISECTION
    • MAX_PERM_CYCLE

  • Keyword descriptions

    • BISECTION
    BISECTION {Integer}
    how many time slices to change at once (+1). Must be a power of 2 currently
    This keyword cannot be repeated and it expects precisely one integer.
    Default value: 8
    • MAX_PERM_CYCLE
    MAX_PERM_CYCLE {Integer}
    how large cyclic permutations to try
    This keyword cannot be repeated and it expects precisely one integer.
    Default value: 6
