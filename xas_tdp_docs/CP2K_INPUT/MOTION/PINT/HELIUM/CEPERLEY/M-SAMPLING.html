Section M-SAMPLING

  • Permutation cycle length sampling settings

  • Keyword descriptions

    • DISTRIBUTION-TYPE
    DISTRIBUTION-TYPE {Keyword}
    Distribution from which the cycle length m is sampled
    This keyword cannot be repeated and it expects precisely one keyword.
    Default value: UNIFORM
    List of valid keywords:
    • EXPONENTIAL
      Without description, yet.
    • GAUSSIAN
      Without description, yet.
    • LINEAR
      Without description, yet.
    • QUADRATIC
      Without description, yet.
    • SINGLEV
      Without description, yet.
    • UNIFORM
      Without description, yet.
    • M-RATIO
    M-RATIO {Real}
    Probability ratio betw M-VALUE and other cycle lengths
    This keyword cannot be repeated and it expects precisely one real.
    Default value: 1.00000000E+000
    • M-VALUE
    M-VALUE {Integer}
    Value of m treated in a special way (specific behavior depends on the distribution type chosen)
    This keyword cannot be repeated and it expects precisely one integer.
    Default value: 1
