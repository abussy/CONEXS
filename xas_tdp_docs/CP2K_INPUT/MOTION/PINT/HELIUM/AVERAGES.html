Section AVERAGES

  • Average properties (used for restarts)
  • Section path: CP2K_INPUT / MOTION / PINT / HELIUM / AVERAGES
  • This section cannot be repeated.

  • Subsections

    • none

  • Keywords

    • IWEIGHT
    • MOMENT_OF_INERTIA
    • PROJECTED_AREA
    • PROJECTED_AREA_2
    • RDF
    • RHO
    • WINDING_NUMBER_2

  • Keyword descriptions

    • IWEIGHT
    IWEIGHT {Integer}
    Weight for the restarted quantities (number of MC steps used to calculate the accumulated averages)
    This keyword cannot be repeated and it expects precisely one integer.
    Default value: 0
    • MOMENT_OF_INERTIA
    MOMENT_OF_INERTIA {Real} ...
    Moment of inertia vector for all environments
    This keyword can be repeated and it expects a list of reals.
    • PROJECTED_AREA
    PROJECTED_AREA {Real} ...
    Projected area vector for all environments
    This keyword can be repeated and it expects a list of reals.
    • PROJECTED_AREA_2
    PROJECTED_AREA_2 {Real} ...
    Projected area vector squared for all environments
    This keyword can be repeated and it expects a list of reals.
    • RDF
    RDF {Real} ...
    Radial distributions averaged over all environments
    This keyword can be repeated and it expects a list of reals.
    • RHO
    RHO {Real} ...
    Spatial distributions averaged over all environments
    This keyword can be repeated and it expects a list of reals.
    • WINDING_NUMBER_2
    WINDING_NUMBER_2 {Real} ...
    Winding number vector squared for all environments
    This keyword can be repeated and it expects a list of reals.
