Section THERMOSTAT_ENERGY

  • Information to initialize the CSVR thermostat energy.
  • Section path: CP2K_INPUT / MOTION / PINT / GLE / THERMOSTAT_ENERGY
  • This section cannot be repeated.

  • Subsections

    • none

  • Keywords

    • DEFAULT_KEYWORD

  • Keyword descriptions

    • DEFAULT_KEYWORD
    Specify an initial thermostat energy for CSVR thermostat.
    This keyword can be repeated and it expects precisely one real.
    Default unit: [internal_cp2k]
