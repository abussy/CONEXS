Section DRIVER

  • This section defines the parameters needed to run in i-PI driver mode.
  • Section path: CP2K_INPUT / MOTION / DRIVER
  • This section cannot be repeated.
  • This section cites the following references: [Ceriotti2014] [Kapil2016]

  • Subsections

    • none

  • Keywords

    • HOST
    • PORT
    • SLEEP_TIME
    • UNIX

  • Keyword descriptions

    • HOST
    HOST {Word}
    Host name for the i-PI server.
    This keyword cannot be repeated and it expects precisely one word.
    Default value: localhost
    • PORT
    PORT {Integer}
    Port number for the i-PI server.
    This keyword cannot be repeated and it expects precisely one integer.
    Default value: 12345
    • SLEEP_TIME
    SLEEP_TIME {Real}
    Sleeping time while waiting for for driver commands [s].
    This keyword cannot be repeated and it expects precisely one real.
    Default value: 1.00000000E-002
    • UNIX
    UNIX {Logical}
    Use a UNIX socket rather than an INET socket.
    This keyword cannot be repeated and it expects precisely one logical.
    The lone keyword behaves as a switch to .TRUE.
    Default value: .FALSE.
