Section EIGENSOLVER

  • Benchmark and test the eigensolver routines.
  • Section path: CP2K_INPUT / TEST / EIGENSOLVER
  • This section can be repeated.

  • Subsections

    • none

  • Keywords

    • DIAG_METHOD
    • EIGENVALUES
    • INIT_METHOD
    • N
    • N_LOOP

  • Keyword descriptions

    • DIAG_METHOD
    DIAG_METHOD {Keyword}
    Diagonalization strategy
    This keyword cannot be repeated and it expects precisely one keyword.
    Default value: SYEVD
    List of valid keywords:
    • SYEVD
      (sca)lapacks syevd
    • SYEVX
      (sca)lapacks syevx
    • EIGENVALUES
    EIGENVALUES {Integer}
    number of eigenvalues to be computed (all=<0)
    This keyword cannot be repeated and it expects precisely one integer.
    Default value: -1
    • INIT_METHOD
    INIT_METHOD {Keyword}
    Initialization approach
    This keyword cannot be repeated and it expects precisely one keyword.
    Default value: RANDOM
    List of valid keywords:
    • RANDOM
      use a random initial matrix
    • READ
      read a matrix from file MATRIX
    • N
    N {Integer}
    Dimension of the square matrix
    This keyword cannot be repeated and it expects precisely one integer.
    Default value: 256
    • N_LOOP
    N_LOOP {Integer}
    Number of operations being timed (useful for small matrices).
    This keyword cannot be repeated and it expects precisely one integer.
    Default value: 10
