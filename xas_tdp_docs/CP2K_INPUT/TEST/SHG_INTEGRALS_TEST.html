Section SHG_INTEGRALS_TEST

  • Parameters for testing the SHG 2-center integrals for different r12 operators. Test w.r.t. performance and accurarcy.
  • Section path: CP2K_INPUT / TEST / SHG_INTEGRALS_TEST
  • This section cannot be repeated.

    • SECTION_PARAMETERS
    &SHG_INTEGRALS_TEST {Logical}
    Controls the activation the SHG integral test.
    This keyword cannot be repeated and it expects precisely one logical.
    The lone keyword behaves as a switch to .TRUE.
    Default value: .FALSE.
    • ABC

    • ACCURACY_LEVEL
    ACCURACY_LEVEL {Real}

    This keyword cannot be repeated and it expects precisely one real.
    Default value: 1.00000000E-008
