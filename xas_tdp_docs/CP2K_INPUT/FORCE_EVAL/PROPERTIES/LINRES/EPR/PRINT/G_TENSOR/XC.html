Section XC

  • Parameters needed for the calculation of the eXchange and Correlation potential
  • Section path: CP2K_INPUT / FORCE_EVAL / PROPERTIES / LINRES / EPR / PRINT / G_TENSOR / XC
  • This section cannot be repeated.

  • Subsections

    • ADIABATIC_RESCALING
    • GCP_POTENTIAL
    • HF
    • VDW_POTENTIAL
    • WF_CORRELATION
    • XC_FUNCTIONAL
    • XC_GRID
    • XC_POTENTIAL

  • Keywords

    • DENSITY_CUTOFF
    • DENSITY_SMOOTH_CUTOFF_RANGE
    • FUNCTIONAL_ROUTINE
    • GRADIENT_CUTOFF
    • TAU_CUTOFF

  • Keyword descriptions

    • DENSITY_CUTOFF
    DENSITY_CUTOFF {Real}
    The cutoff on the density used by the xc calculation
    This keyword cannot be repeated and it expects precisely one real.
    Default value: 1.00000000E-010
    • DENSITY_SMOOTH_CUTOFF_RANGE
    DENSITY_SMOOTH_CUTOFF_RANGE {Real}
    Parameter for the smoothing procedure inxc calculation
    This keyword cannot be repeated and it expects precisely one real.
    Default value: 0.00000000E+000
    • FUNCTIONAL_ROUTINE
    FUNCTIONAL_ROUTINE {Keyword}
    Select the code for xc calculation
    This keyword cannot be repeated and it expects precisely one keyword.
    Default value: NEW
    List of valid keywords:
    • DEBUG
      Use debug new code for exchange-correlation functional calculation
    • NEW
      Use new code for exchange-correlation functional calculation
    • OLD
      Use old code for exchange-correlation functional calculation
    • TEST_LSD
      Use test local-spin-density approximation code for exchange-correlation functional calculation
    • GRADIENT_CUTOFF
    GRADIENT_CUTOFF {Real}
    The cutoff on the gradient of the density used by the xc calculation
    This keyword cannot be repeated and it expects precisely one real.
    Default value: 1.00000000E-010
    • TAU_CUTOFF
    TAU_CUTOFF {Real}
    The cutoff on tau used by the xc calculation
    This keyword cannot be repeated and it expects precisely one real.
    Default value: 1.00000000E-010
