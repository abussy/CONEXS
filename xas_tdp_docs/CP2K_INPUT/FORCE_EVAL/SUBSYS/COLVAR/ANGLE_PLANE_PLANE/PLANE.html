Section PLANE

  • This section defines the plane. When using this colvar, two plane section must be defined!
  • Section path: CP2K_INPUT / FORCE_EVAL / SUBSYS / COLVAR / ANGLE_PLANE_PLANE / PLANE
  • This section can be repeated.

  • Subsections

    • none

  • Keywords

    • ATOMS
    • DEF_TYPE
    • NORMAL_VECTOR

  • Keyword descriptions

    • ATOMS
    ATOMS {Integer} {Integer} {Integer}
    Specifies the indexes of 3 atoms/points defining the plane.
    This keyword cannot be repeated and it expects precisely 3 integers.
    • DEF_TYPE
    DEF_TYPE {Keyword}
    Specify how the plane is defined: either by 3 atoms or by a fixed normal vector. At least one plane must be defined through atoms.
    This keyword cannot be repeated and it expects precisely one keyword.
    Default value: ATOMS
    List of valid keywords:
    • ATOMS
      Plane defined by the position of 3 atoms
    • VECTOR
      Plane defined by a fixed normal vector
    • NORMAL_VECTOR
    NORMAL_VECTOR {Real} {Real} {Real}
    Alternatively to 3 atoms/points one can define one of the two, planes by defining its NORMAL vector.
    This keyword cannot be repeated and it expects precisely 3 reals.
