Section WAVELET

  • Sets up parameters of wavelet based poisson solver.This solver allows for non-periodic (PERIODIC NONE) boundary conditions and slab-boundary conditions (but only PERIODIC XZ).It does not require very large unit cells, only that the density goes to zero on the faces of the cell.The use of PREFERRED_FFT_LIBRARY FFTSG is required
  • Section path: CP2K_INPUT / FORCE_EVAL / QMMM / PERIODIC / POISSON / WAVELET
  • This section cannot be repeated.
  • This section cites the following references: [Genovese2006] [Genovese2007]

  • Subsections

    • none

  • Keyword descriptions

    • SCF_TYPE
    SCF_TYPE {Integer}
    Type of scaling function used in the wavelet approach, the total energy depends on this choice,and the convergence with respect to cutoff depends on the selected scaling functions.Possible values are 8,14,16,20,24,30,40,50,60,100
    This keyword cannot be repeated and it expects precisely one integer.
    Default value: 40
