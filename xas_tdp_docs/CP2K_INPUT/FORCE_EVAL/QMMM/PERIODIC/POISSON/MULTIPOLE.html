Section MULTIPOLE

  • This section is used to set up the decoupling of QM periodic images with the use of density derived atomic point charges.
  • Section path: CP2K_INPUT / FORCE_EVAL / QMMM / PERIODIC / POISSON / MULTIPOLE
  • This section cannot be repeated.

  • Keyword descriptions

    • ANALYTICAL_GTERM
    ANALYTICAL_GTERM {Logical}
    Evaluates the Gterm in the Ewald Scheme analytically instead of using Splines.
    This keyword cannot be repeated and it expects precisely one logical.
    The lone keyword behaves as a switch to .TRUE.
    Default value: .FALSE.
    • EWALD_PRECISION
    EWALD_PRECISION {Real}
    Precision achieved in the Ewald sum.
    This keyword cannot be repeated and it expects precisely one real.
    Default value: 1.00000000E-006
    Default unit: [hartree]
    • NGRIDS
    NGRIDS {Integer} {Integer} {Integer}
    Specifies the number of grid points used for the Interpolation of the G-space term
    This keyword cannot be repeated and it expects precisely 3 integers.
    Default values: 50 50 50
    • RCUT
    RCUT {Real}
    Real space cutoff for the Ewald sum.
    This keyword cannot be repeated and it expects precisely one real.
    Default unit: [angstrom]
