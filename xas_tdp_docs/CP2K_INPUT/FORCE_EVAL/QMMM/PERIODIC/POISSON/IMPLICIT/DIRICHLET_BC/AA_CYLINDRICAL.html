Section AA_CYLINDRICAL

  • Parameters for creating axis-aligned cylindrical Dirichlet boundary regions.
  • Section path: CP2K_INPUT / FORCE_EVAL / QMMM / PERIODIC / POISSON / IMPLICIT / DIRICHLET_BC / AA_CYLINDRICAL

  • Keyword descriptions

    • APX_TYPE
    APX_TYPE {Keyword}
    Specifies the type of the n-gonal prism approximating the cylinder.
    This keyword cannot be repeated and it expects precisely one keyword.
    Default value: CIRCUMSCRIBED
    List of valid keywords:
    • CIRCUMSCRIBED
      Without description, yet.
    • INSCRIBED
      Without description, yet.
    • BASE_CENTER
    BASE_CENTER {Real} {Real}
    The y and z coordinates (x and z or x and y coordinates, depending on the choice of the parallel axis) of the cylinder's base center.
    This keyword cannot be repeated and it expects precisely 2 reals.
    Default unit: [angstrom]
    • BASE_RADIUS
    BASE_RADIUS {Real}
    The base radius of the cylinder.
    This keyword cannot be repeated and it expects precisely one real.
    Default value: 1.00000000E+000
    Default unit: [angstrom]
    • DELTA_ALPHA
    DELTA_ALPHA {Real}
    A central angle specifying the gap between the faces of the n-gonal prism. To avoide overlap between the cuboids (of the given thickness) built on top of the faces, a larger value is required if the number of faces (N_SIDES) is quite few and/or the base radius is fairly small.
    This keyword cannot be repeated and it expects precisely one real.
    Default value: 5.00000000E-002
    Default unit: [rad]
    • FREQUENCY
    FREQUENCY {Real}
    The frequency with which the oscillating fraction oscillates.
    This keyword cannot be repeated and it expects precisely one real.
    Default value: 0.00000000E+000
    Default unit: [s^-1]
    • N_PRTN
    N_PRTN {Integer} {Integer}
    The number of partitions along the face edges of the prism for tiling. If the edges have different lengths, from the two given values, the larger one will be assigned to the longer edge.
    This keyword cannot be repeated and it expects precisely 2 integers.
    Default values: 1 1
    • N_SIDES
    N_SIDES {Integer}
    The number of sides (faces) of the n-gonal prism approximating the cylinder.
    This keyword cannot be repeated and it expects precisely one integer.
    Default value: 5
    • OSCILLATING_FRACTION
    OSCILLATING_FRACTION {Real}
    A fraction of the field can be set to oscilate over time.
    This keyword cannot be repeated and it expects precisely one real.
    Default value: 0.00000000E+000
    • PARALLEL_AXIS
    PARALLEL_AXIS {Keyword}
    The coordinate axis that the cylindrical region extends along.
    This keyword cannot be repeated and it expects precisely one keyword.
    List of valid keywords:
    • X
      Without description, yet.
    • Y
      Without description, yet.
    • Z
      Without description, yet.
    • PHASE
    PHASE {Real}
    The phase of the oscillattion. A phase of zero corresponds to a cosine function.
    This keyword cannot be repeated and it expects precisely one real.
    Default value: 0.00000000E+000
    • SMOOTHING_WIDTH
    SMOOTHING_WIDTH {Real}
    The width of the transition region between the Dirichlet subdomain and its surrounding.
    This keyword cannot be repeated and it expects precisely one real.
    Default value: 2.00000000E-001
    Default unit: [angstrom]
    Alias names for this keyword: SIGMA
    • THICKNESS
    THICKNESS {Real}
    The thickness of the cylinder.
    This keyword cannot be repeated and it expects precisely one real.
    Default value: 7.50000000E-001
    Default unit: [angstrom]
    • V_D
    V_D {Real}
    The value of the fixed potential to be imposed at the cylindrical Dirichlet boundary.
    This keyword cannot be repeated and it expects precisely one real.
    Default unit: [volt]
    • XTNT
    XTNT {Real} {Real}
    The extents of the cylinder along its central axis.
    This keyword cannot be repeated and it expects precisely 2 reals.
    Default unit: [angstrom]
