Section PRINT

  • Collects all printing options related to the Wannier centers and properties computed with Wannier centers.
  • Section path: CP2K_INPUT / FORCE_EVAL / DFT / PRINT / ACTIVE_SPACE / LOCALIZE / PRINT
  • This section cannot be repeated.

  • Subsections

    • LOC_RESTART
    • MOLECULAR_DIPOLES
    • MOLECULAR_STATES
    • PROGRAM_RUN_INFO
    • TOTAL_DIPOLE
    • WANNIER_CENTERS
    • WANNIER_CUBES
    • WANNIER_SPREADS
    • WANNIER_STATES
