Section OPT_EMBED

  • Keyword descriptions

    • CHARGE_DISTR_WIDTH
    CHARGE_DISTR_WIDTH {Real}
    Width of the Gaussian representing point charges. To be used with ADD_COULOMB_POT.
    This keyword cannot be repeated and it expects precisely one real.
    Default value: 1.12490000E+000
    • DENS_CONV_INT
    DENS_CONV_INT {Real}
    Convergence criterion for the integrated electron density difference.
    This keyword cannot be repeated and it expects precisely one real.
    Default value: 1.00000000E-001
    • DENS_CONV_MAX
    DENS_CONV_MAX {Real}
    Convergence criterion for the maximum electron density difference.
    This keyword cannot be repeated and it expects precisely one real.
    Default value: 1.00000000E-002
    • EMBED_CUBE_FILE_NAME
    EMBED_CUBE_FILE_NAME {String}
    Root of the file name where to read the embedding potential (guess) as a cube.
    This keyword cannot be repeated and it expects precisely one string.
    • EMBED_RESTART_FILE_NAME
    EMBED_RESTART_FILE_NAME {String}
    Root of the file name where to read the embedding potential guess.
    This keyword cannot be repeated and it expects precisely one string.
    • EMBED_SPIN_CUBE_FILE_NAME
    EMBED_SPIN_CUBE_FILE_NAME {String}
    Root of the file name where to read the spin part of the embedding potential (guess) as a cube.
    This keyword cannot be repeated and it expects precisely one string.
    • FAB
    FAB {Logical}
    Finzel-Ayers-Bultinck iterative update. Generally, not reliable.
    This keyword cannot be repeated and it expects precisely one logical.
    Default value: .FALSE.
    • GRID_OPT
    GRID_OPT {Logical}
    Optimize embedding potential on the grid.
    This keyword cannot be repeated and it expects precisely one logical.
    Default value: .TRUE.
    • LEEUWEN-BAERENDS
    LEEUWEN-BAERENDS {Logical}
    Van Leeuwen-Baerends iterative update. Alternative to Wu-Yang optimizer. Use only with ADD_CONTST_POT.
    This keyword cannot be repeated and it expects precisely one logical.
    Default value: .FALSE.
    • N_ITER
    N_ITER {Integer}
    Maximum number of iterations in the optimization procedure.
    This keyword cannot be repeated and it expects precisely one integer.
    Default value: 50
    • OPTIMIZER
    OPTIMIZER {Keyword}
    Optimize embedding potential.
    This keyword cannot be repeated and it expects precisely one keyword.
    Default value: STEEPEST_DESCENT
    List of valid keywords:
    • LEVEL_SHIFT
      Level shift.
    • QUASI_NEWTON
      Quasi-Newton.
    • STEEPEST_DESCENT
      Steepest descent.
    • POT_GUESS
    POT_GUESS {Keyword}
    Specifies the guess of the embedding potential. For optimization in finite basis (not grid optimization) in is a constant part to be added to the one in finite basis.
    This keyword cannot be repeated and it expects precisely one keyword.
    Default value: NONE
    List of valid keywords:
    • DIFF
      Initial density difference. A euristic but working approach.
    • FERMI_AMALDI
      Fermi-Amaldi potential. More rigorous than DIFF, although less efficient.
    • NONE
      Initial guess is zero grid.
    • RESP
      Coulomb interaction between the subsystem using RESP charges) on the total system.
    • READ_EMBED_POT
    READ_EMBED_POT {Logical}
    Read the embedding potential restart vector as a guess.
    This keyword cannot be repeated and it expects precisely one logical.
    Default value: .FALSE.
    • READ_EMBED_POT_CUBE
    READ_EMBED_POT_CUBE {Logical}
    Read the embedding potential (restart) from the cube file.
    This keyword cannot be repeated and it expects precisely one logical.
    Default value: .FALSE.

    This keyword cannot be repeated and it expects precisely one real.

    • VW_CUTOFF
    VW_CUTOFF {Real}

    This keyword cannot be repeated and it expects precisely one real.
    Default value: 1.00000000E-002
