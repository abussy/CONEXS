Section PLANAR

  • Parameters for creating planar (rectangular) Dirichlet boundary regions with given vertices.
  • Section path: CP2K_INPUT / FORCE_EVAL / DFT / POISSON / IMPLICIT / DIRICHLET_BC / PLANAR
  • This section can be repeated.

  • Subsections

    • none

  • Keyword descriptions

    • A
    A {Real} {Real} {Real}
    Coordinates of the vertex A.
    This keyword cannot be repeated and it expects precisely 3 reals.
    Default unit: [angstrom]
    • B
    B {Real} {Real} {Real}
    Coordinates of the vertex B.
    This keyword cannot be repeated and it expects precisely 3 reals.
    Default unit: [angstrom]
    • C
    C {Real} {Real} {Real}
    Coordinates of the vertex C.
    This keyword cannot be repeated and it expects precisely 3 reals.
    Default unit: [angstrom]
    • FREQUENCY
    FREQUENCY {Real}
    The frequency with which the oscillating fraction oscillates.
    This keyword cannot be repeated and it expects precisely one real.
    Default value: 0.00000000E+000
    Default unit: [s^-1]
    • N_PRTN
    N_PRTN {Integer} {Integer}
    The number of partitions along the edges for tiling the rectangular region. If the edges have different lengths, from the two given values, the larger one will be assigned to the longer edge.
    This keyword cannot be repeated and it expects precisely 2 integers.
    Default values: 1 1
    • OSCILLATING_FRACTION
    OSCILLATING_FRACTION {Real}
    A fraction of the field can be set to oscilate over time.
    This keyword cannot be repeated and it expects precisely one real.
    Default value: 0.00000000E+000
    • PHASE
    PHASE {Real}
    The phase of the oscillattion. A phase of zero corresponds to a cosine function.
    This keyword cannot be repeated and it expects precisely one real.
    Default value: 0.00000000E+000
    • SMOOTHING_WIDTH
    SMOOTHING_WIDTH {Real}
    The width of the transition region between the Dirichlet subdomain and its surrounding.
    This keyword cannot be repeated and it expects precisely one real.
    Default value: 2.00000000E-001
    Default unit: [angstrom]
    Alias names for this keyword: SIGMA
    • THICKNESS
    THICKNESS {Real}
    The thickness of the planar Dirichlet region.
    This keyword cannot be repeated and it expects precisely one real.
    Default value: 7.50000000E-001
    Default unit: [angstrom]
    • V_D
    V_D {Real}
    The value of the fixed potential to be imposed at the planar Dirichlet boundary.
    This keyword cannot be repeated and it expects precisely one real.
    Default unit: [volt]
